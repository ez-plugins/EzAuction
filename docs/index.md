# EzAuction Documentation

A feature-rich, modern auction house plugin for **Paper 1.21+** servers. EzAuction provides a polished GUI, buy orders, live auctions, team-scoped listings, Discord webhooks, hologram displays, and both YAML and MySQL storage backends.

## What you will find here

- Fast installation and first-start checklist
- Full configuration reference for every YAML file
- Command and permission tables
- Feature walkthroughs (auction house, buy orders, live auctions, team auctions)
- Integration guides (Discord, EzShops, TeamsAPI)
- Developer API and custom event reference

## Recommended onboarding path

1. Read [Getting Started](getting-started.md)
2. Complete [Installation](installation.md)
3. Apply [Configuration](configuration/)
4. Review [Features](features/) for anything you want to enable
5. Set up [Integrations](integrations/) as needed
6. Use [Reference](reference/) for exact commands, permissions, and events
