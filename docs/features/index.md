# Features

This section documents each EzAuction feature individually. Each page covers what the feature does, how to enable it, key configuration options, and relevant commands.

## Feature overview

| Feature | Description |
|---------|-------------|
| [Auction House](auction-house.md) | Core browsing, buying, and searching GUI |
| [Sell Menu](sell-menu.md) | Item listing GUI with price and amount controls |
| [Buy Orders](buy-orders.md) | Players post purchase offers; sellers fulfil them |
| [Live Auctions](live-auctions.md) | Real-time bidding queue |
| [Team Auctions](team-auctions.md) | Team-scoped listings visible only to teammates |
| [Transaction History](transaction-history.md) | Per-player history of all buys and sells |
